The Dark/Light Mode Toggler

Let's step it up to using a boolean (true/false) switch to change how your page looks.

What to do:

    Open a blank file.

    Create a boolean state variable (e.g., isDark) that starts as false.

    Add a button that says "Toggle Theme". Clicking this button should flip the boolean state to its opposite value every time.

    Use a wrapper <div> around your content. Use an inline style attribute on that <div> to dynamically change its background color and text color based on your boolean state (e.g., if true: black background & white text; if false: white background & black text).
